TRANSFERENCIA

DATOS PERSONALES

Pon un nombre.

Minimo 3 letras

Ingresa el IBAN.

El IBAN debe tener al menos
15 caracteres.

El IBAN no puede tener más de 34 caracteres.

Ingresa el BIC.

El BIC debe tener al menos 8 caracteres.

El BIC no puede tener más de 11 caracteres.

DATOS DE TU CUENTA

Pon un nombre.

Minimo 3 letras

Ingresa la fecha de caducidad.

Formato inválido. Usa MM/AA.

Ingresa el CVV.

El CVV debe tener exactamente 3 dígitos.

El CVV no puede tener más de 3 dígitos.
